# Cloud-Vault 记账软件 需求分析

## 一、项目概述

Cloud-Vault 是一款基于 Cloudflare 全家桶的云端记账软件，支持 Web 端和移动端，主打高质感 UI 与流畅体验。通过 AI 辅助实现智能记账，降低用户手动录入成本。

---

## 二、用户角色

| 角色 | 说明 |
|------|------|
| 系统管理员 | 部署后的初始管理员，负责用户、邀请码和系统设置管理 |
| 普通用户 | 日常记账、查看报表、管理个人财务 |
| 家庭/团队用户 | 多人共享账本，协同记账（后期扩展） |

## 三、功能需求

### 3.1 用户系统（P0）

| 编号 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| U-01 | 注册/登录 | 邮箱注册、密码登录，支持 OAuth（Google/GitHub）；注册必须通过系统邀请码校验 | P0 |
| U-02 | 用户信息管理 | 头像、昵称、默认货币设置 | P0 |
| U-03 | 安全设置 | 修改密码、两步验证（预留） | P1 |
| U-04 | 账号注销 | 删除账户及所有关联数据 | P2 |
| U-05 | 注册邀请码 | 注册或 OAuth 首次绑定账号时必须填写管理员分发的邀请码；邀请码支持次数、有效期、启用/禁用和使用记录 | P0 |
| U-06 | 初始管理员 | 部署时配置管理员邮箱和初始化邀请码，首次使用该邮箱和邀请码注册后成为系统管理员 | P0 |

| B-03 | 收入/支出/转账 | 三种账单类型，转账支持跨账户 | P0 |
| B-04 | 账单列表 | 按时间分组展示，支持筛选和搜索 | P0 |
| B-05 | 批量操作 | 批量删除、批量修改分类 | P1 |
| B-06 | 周期性账单 | 设置固定周期支出（房租、订阅等），自动生成 | P1 |
| B-07 | 退款记录 | 关联原账单，记录退款信息 | P2 |

## 四、非功能需求

### 4.1 性能

- 页面首屏加载 < 1.5s（利用 Cloudflare 边缘节点）
- 账单列表滚动流畅，支持虚拟滚动（大数据量）
- AI 解析响应 < 2s

### 4.2 安全

- 所有 API 走 HTTPS
- 用户密码 bcrypt 哈希存储
- 注册邀请码只存哈希，不在数据库保存明文
- 默认关闭公开注册，注册必须提供有效邀请码
- JWT Token 认证，存储于 KV
- 敏感操作（删除、导出）需二次验证
- SQL 注入防护（D1 参数化查询）

### 4.3 可用性

- 移动端优先的响应式设计
- 断网时可查看已缓存数据（PWA 离线能力）
- 操作可撤销（误删账单 30s 内可恢复）

### 4.4 兼容性

- Web：Chrome、Safari、Firefox、Edge 最新两个大版本
- 移动端：iOS 15+、Android 12+

### 4.5 可扩展性

- 模块化架构，功能可按需加载
- 预留 R2 存储接口用于发票/收据图片
- 预留共享账本的多用户权限模型


## 六、数据模型概览

| 实体 | 核心字段 | 说明 |
|------|---------|------|
| User | id, email, password_hash, nickname, avatar, currency, system_role, registration_invite_id | 用户 |
| RegistrationInvite | id, code_hash, created_by, allowed_email, max_uses, used_count, expires_at, status | 注册邀请码 |
| Book | id, owner_user_id, name, type, default_currency | 账本 |
| Account | id, book_id, name, type, balance, icon | 账户 |
| Category | id, book_id, name, type(income/expense), icon, parent_id | 分类 |
| Transaction | id, book_id, account_id, category_id, type, amount, note, date, tags | 账单 |
| Budget | id, book_id, category_id, amount, period(monthly) | 预算 |
| RecurringRule | id, book_id, template, frequency, next_date | 周期账单规则 |
